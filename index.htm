Episode 1: a wild grid appears

I'm making a game! The goal: a realtime-ish roguelike. Things I'd like to include:

  • Lighting (directional/coloured)
  • RPG elements: inventory, conversations
  • Party system
  • Magic system
  • Various AI techniques: pathfinding, behaviour trees, knowledge sharing
  • Procedural generation

Tonight I've started off with a random "tilemap" generator, basic keyboard movement and a renderer. Not much to write home about here; most of the time was actually spent writing a some Ruby to auto generate the website from specified git commits.

Next up: basic map generation, wall collision detection.
